Design Innovation Project

Crowd Counting

Monitoring the Crowd of People by Deep Learning Enabled Visual Analytics

The purpose of this project is to use deep learning enabled visual analytics to monitor the crowd of people and improve the accuracy and localization of the crowd count in a specific area. The team will be working with the pre-trained models and existing dataset and performing visual analytics by using Convolution Neural Network (CNN) Method.

● The team will design a simple user interface that allows the user to input the data (image).

● We will be exploring the principles of at least one deep learning based crowd counting method to work on the dataset.

● The output will be a density map showing the crowdedness of an area.

● Model will be used to produce a density map to show the crowding/ concentration.

High-level Project and Product Requirements:

Using the data, we aim to train on the existing neural network models by using CNN and Tensorflow/PyTorch. Implementing more features such as headcounting. Design a GUI to enable users to upload their images on a web-based application which will be compatible on different browsers and devices.
